## Option 3: Composer

Composer is a dependency manager for PHP. Used as a way to access and include third-party code in your programs and on your site.

Typically used in the command line to install your dependencies.

Learn more about composer here: 

* [What is PHP Composer](http://culttt.com/2013/01/07/what-is-php-composer/)
* [Official Composer website](https://getcomposer.org/)
* [WordPress Packagist](http://wpackagist.org/) - Install WordPress plugins and themes using Composer

Note:
- Composer is a dependency manager for PHP. 
- Used on the command line to install third-party dependencies that your plugin or theme uses. Including what dependencies your site uses as a whole.
